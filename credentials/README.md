# Credentials format

## Algorithmia

File: `algorithmia.json`

```json
{
  "apiKey": "INSERT_THE_KEY_HERE"
}
```

## Watson Natural Language Understanding

File: `watson-nlu.json`

```json
{
  "apikey": "...",
  "iam_apikey_description": "...",
  "iam_apikey_name": "...",
  "iam_role_crn": "...",
  "iam_serviceid_crn": "...",
  "url": "..."
}
```
